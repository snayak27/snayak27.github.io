"Imagine hiring a night-shift employee who shows up every day the moment your computer starts — knows your preferences, speaks three expert languages, and never takes a break."

Claude Code is powerful but interactive — it works when you're at your computer. What about tasks that should run on a schedule? Or tools that need to stay running all day?

Hermes is the always-on counterpart. It starts at login, runs silently in WSL, and has custom skills for equity research, stealth scraping, bulk crawling, and repo integration. Switch personality with one command.

In technical terms: NousResearch Hermes agent framework running via uv in WSL2, registered as a Windows Task Scheduler logon trigger, with OpenRouter smart model routing.
