The glossary

Heard a computer word and not sure what it means? Here is every term used in this guide, explained in a sentence or two, with the floor it lives on. Tip: press / anywhere to search the whole guide.

Floor 5
Algorithm
A precise step-by-step recipe for solving a problem. Good algorithms stay fast even when the data is huge.
Floor 3
Assembly language
A low-level programming language that maps directly to the CPU's raw binary instructions, allowing programmers to write human-readable commands instead of zeroes and ones.
Floor 1
Attention / Transformer
The 2017 neural-network mathematical design that reads a whole sequence at once using parallel linear algebra. The basis of every modern AI chatbot.
Floor 5
Big-O notation
A shorthand for how an algorithm's time or memory grows as its input grows — the difference between scaling to millions of users and grinding to a halt.
Floor 1
Bit
The smallest unit of information — a single 0 or 1. Named by Claude Shannon in 1948. Everything digital is built from bits.
Floor 4
Cloud
Renting computing power and storage from large remote data centres instead of owning the hardware yourself.
Floor 2
CPU
The central processing unit — a chip with a few powerful cores that handle general tasks one after another. The classic "brain" of a computer.
Floor 3
CUDA
NVIDIA's software (2007) that lets ordinary programs run on thousands of GPU cores — the bridge that later let AI models train on GPUs.
Floor 6
Deep learning
A subset of machine learning using neural networks with many hidden layers ("deep" stacks). It allows models to automatically extract features from complex, raw data.
Floor 4
DNS
The Domain Name System (1983) — the internet's phone book, translating names like youtube.com into the numeric addresses computers use.
Floor 2
GPU
A graphics processing unit — a chip with thousands of simple cores doing the same math at once. Built for games, now the engine of AI.
Floor 7
GUI
Graphical User Interface — a visual system of windows, icons, buttons and mouse pointers (pioneered by Xerox Alto and popularized by Apple's Macintosh) replacing command-line text.
Floor 1
Halting problem
Alan Turing's 1936 mathematical proof that no general algorithm can predict whether any program will eventually stop or run forever — the first definition of computing's absolute limits.
Floor 4
HTTP
The shared set of rules for requesting and sending web pages, images, and data across the internet. The language that makes the World Wide Web work.
Floor 1
Information theory
Claude Shannon's 1948 mathematics of measuring, compressing and transmitting information. The foundation of all digital communication.
Floor 2
Integrated circuit (IC)
Fabricating an entire electrical circuit — transistors, resistors and wiring — on a single tiny slice of silicon. It allowed computers to shrink from room-sized cabinets to pocket-sized devices.
Floor 6
Large Language Model (LLM)
A very large Transformer trained on huge amounts of text — like ChatGPT, Claude or Gemini — that can write, answer and converse.
Floor 1
Linear algebra
The mathematics of combining grids of numbers (vectors and matrices). It is what AI models actually compute, and what GPUs are built to do fast.
Floor 6
Machine learning
Teaching a computer by showing it many examples and letting it find the rules itself, rather than coding the rules by hand.
Floor 2
Microprocessor
A whole CPU on a single chip. The first commercial one was the Intel 4004 in 1971 — it put computers into everyday devices.
Floor 6
Neural network
A web of simple numeric units, loosely inspired by brain cells, whose connections are tuned by training until it makes good predictions.
Floor 3
Operating system
The manager (Windows, macOS, Android, Linux) that shares one computer between many programs and keeps them from interfering.
Floor 4
Packet switching
Chopping a message into small numbered packets that travel independently and are reassembled at the destination. The core idea of the internet.
Floor 1
P vs NP
The famous unsolved question of whether every problem that's quick to check is also quick to solve. Stated by Stephen Cook in 1971; a $1,000,000 prize.
Floor 4
Protocol
A shared set of rules that computers agree to follow so they can understand each other — like TCP/IP, the internet's common language.
Floor 5
Relational database
Storing data in linked tables, an idea from Edgar Codd in 1970. The basis of SQL and most business data systems.
Floor 7
Responsive design
Building an app or page that rearranges itself to fit any screen size — one app that works on phone, tablet and laptop.
Floor 5
SQL
Structured Query Language — the universal coding language used to ask questions, filter records, and update tables inside a relational database.
Floor 3
Stored-program
Keeping a computer's instructions in memory alongside its data, so it can be reprogrammed by loading new instructions. The von Neumann idea (1945).
Floor 4
TCP/IP
The universal language of the internet (adopted 1983) — it gives every machine an address and a reliable way to exchange packets.
Floor 2
Transistor
A tiny solid-state electrical switch, invented in 1947. Billions of them on a chip make up every modern processor.
Floor 1
Turing machine
Alan Turing's 1936 abstract model of computation — the blueprint that every real computer follows, written before any existed.
