The SAP jargon buster

SAP loves acronyms. Here's every one used across this guide, each explained in a single plain-English sentence. Type to filter.

ERPEnterprise Resource Planning
One central system that runs a company's core operations — finance, supply chain, HR — instead of many disconnected tools.
S/4HANA
SAP's flagship, current-generation ERP. "S/4" = 4th generation; "HANA" = the in-memory database it runs on.
HANA
SAP's in-memory database — it keeps data in RAM for real-time speed. Also available as HANA Cloud on BTP.
ECCERP Central Component
The older, legacy generation of SAP ERP. Mainstream support ends in 2027 (2030 with extra fees) — driving a huge migration wave.
BTPBusiness Technology Platform
SAP's platform for integration, app-building, data and AI. Where all custom work lives. Deep-dive →
RISE with SAP
A bundle for moving to private cloud — S/4HANA Private Edition run by SAP. For existing customers migrating off on-premise.
GROW with SAP
A bundle for public cloud — standardized S/4HANA Public Edition, live in weeks. For companies new to SAP Cloud ERP.
On-Premise
Running SAP on the company's own servers, in its own data center. Maximum control, maximum responsibility. More →
Hybrid
A real-world mix of deployment models, stitched together with BTP. The shape of almost every large landscape. More →
Clean Core
The rule: never modify the ERP core; build all custom work as extensions on BTP, so upgrades stay easy and AI stays reliable.
Fit-to-Standard
Adapting your business to SAP's proven best-practice process, rather than customizing SAP to fit your old way.
SAP Activate
SAP's cloud adoption methodology with ready-made best practices. Six phases: Discover → Prepare → Explore → Realize → Deploy → Run.
Fiori
SAP's modern app interface — clean, tile-based screens instead of the old grey transaction-code screens.
Universal Journal
One single table in S/4HANA holding all financial postings — financial and management accounting unified, no reconciliation.
FICO
Financial Accounting (FI) + Controlling (CO) — SAP's finance module set. The domain behind the CFO view.
SAP Build
The low-code/no-code part of BTP — Build Apps, Build Process Automation, Build Work Zone, Build Code.
Build Apps
Drag-and-drop visual app builder on BTP (formerly AppGyver). Build web & mobile apps without traditional coding.
Integration Suite
The BTP tool that connects SAP↔SAP and SAP↔non-SAP systems in real time. An "iPaaS".
iPaaSIntegration Platform as a Service
A cloud service for connecting different software systems together — SAP's is the Integration Suite.
CAP / RAP
SAP's pro-code frameworks — CAP (Cloud Application Programming) and RAP (RESTful ABAP Programming) for building apps the clean-core way.
Joule
SAP's AI copilot, embedded across every app — ask questions in plain language. Deep-dive →
Joule Studio
The BTP tool for building and governing your own AI agents, apps and workflows.
Autonomous Suite
SAP's 2026 fleet of 200+ AI agents and 50+ assistants that run tasks across Finance, Spend, Supply Chain, HCM and CX.
Agent / Assistant
An agent does one precise task; an assistant is mapped to a role and orchestrates several agents to complete a process.
Knowledge Graph
SAP's semantic map of all business entities and relationships — gives AI agents context so they don't hallucinate.
Business Data Cloud
SAP's unified, governed data layer (Datasphere + SAP Analytics Cloud) that makes all data AI-ready.
SuccessFactors
SAP's cloud HR suite — the whole employee journey, hire to retire. Deep-dive →
Ariba
SAP's procurement / source-to-pay suite and supplier network. Deep-dive →
Concur
SAP's travel & expense management product. Deep-dive →
Fieldglass
SAP's system for managing the external workforce — contractors, temps, SOW-based services.
Signavio
SAP's process transformation suite — model processes and mine event logs to find what's broken. Deep-dive →
LeanIX
SAP's enterprise architecture tool — maps the whole IT application landscape. Also powers the AI Agent Hub.
Process Mining
Reconstructing how a process actually ran by reading system event logs — revealing the real flow, not the assumed one.
SAP CX
SAP's customer experience suite — commerce, marketing, sales, service. Deep-dive →
IBPIntegrated Business Planning
SAP's cloud supply chain planning hub — demand forecasting, S&OP, inventory optimization. Deep-dive →
S&OP
Sales & Operations Planning — the process of aligning demand forecasts with supply capacity.
MESManufacturing Execution System
Software that controls and monitors what happens on the factory floor. SAP's is Digital Manufacturing.
EWM / TM
Extended Warehouse Management (warehouse operations) and Transportation Management (freight & logistics).
Two-Tier ERP
Headquarters runs a heavy ERP (e.g. RISE) while smaller subsidiaries run a lighter one (e.g. GROW), connected together.
Business One
SAP's affordable all-in-one ERP for small businesses. More →
MCP / A2A
Open AI standards — Model Context Protocol (tools for agents) and Agent-to-Agent (agents talking to each other). Joule supports both.
