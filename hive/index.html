What I built

Six things that make this system special

⚡
Self-bootstrapping in one script
bootstrap.ps1 detects Windows paths, configures all MCPs, registers Task Scheduler, and launches every service automatically.
👁️
Ring 4: BraveBridge fully containerized
BraveBridge (live Brave browser access) now runs in Docker — hardened Dockerfile, no more host VBS startup script. Cold-boot clean, zero manual steps after restart.
🧠
34 tools, one connection point
Claude connects to one FastMCP gateway. The gateway discovers and proxies all 34 tools with correct type signatures per tool.
🔁
Auto-recovery after restart
Health loop detects when a service comes back online and automatically re-discovers its tools. No manual intervention.
🚫
No Ollama — embeddings via OpenRouter
All vector embeddings use qwen3-embedding-8b via OpenRouter. Saves 8 GB RAM, eliminates a heavy dependency.
✅
CI validates every commit
GitHub Actions checks JSON/YAML configs, frontmatter, secret scan, and presence of all 24 required files on every push.
