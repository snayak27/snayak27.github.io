"Imagine a hospital that automatically sends you to the best specialist — a brain surgeon for brain problems, a heart surgeon for heart problems. Not the same doctor for everything."

Most people use one AI model for everything. It's like going to a general practitioner for brain surgery. You're not getting the best expert for the job.

Claw Router is a smart dispatcher. Tell it the task type — coding, writing, reasoning — and it picks the model with the highest benchmark score for that exact task, within a sensible budget.

In technical terms: Python script → OpenRouter API. Task type → benchmark lookup → best model selection. Critic-revise loops, parallel candidates, chain mode for long documents.

Invocation modes

Four ways to call the router depending on quality needs

single
Single call
One model, one pass. Best for straightforward generation where speed matters.
~$0.001 typical
--revise 2
Critic-revise loop
Writer generates → critic reviews → writer revises. Catches errors automatically.
~$0.004 typical
--chain
Sequential chain
Outline → expand → verify. Each stage feeds the next. Best for long structured docs.
~$0.007 typical
--compete 3
Parallel competition
3 candidates run in parallel → judge picks winner. Best for CVs, proposals, formal reports.
~$0.012 typical

How it works

Task type is detected, benchmarked against the routing table, and sent to the cheapest model that wins on score — never the other way round.
